## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# build app
$ npm run build

# build with watch
$ npm run build:watch

# dev mode
$ npm run start:dev

# dev mode with watch
$ npm run start:watch
```

## Deploy

```bash
# deploy to aws lambda PROD
npm run deploy

# deploy to aws lambda TEST
npm run deploy:test
```
